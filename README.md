## 🛡️ Security & Trust

### Why does Google flag this app?
When installing AnkiVoice Agent, you may see a "Malicious App" or "Play Protect" warning. **This is standard behavior** for experimental open-source apps installed outside the Play Store.

- **Audio Permission**: Because we use a voice-driven AI, the app requires microphone access. Google automatically flags non-store apps that record audio.
- **Local AI**: We bundle the AI engines directly in the APK. This makes the file larger and more complex, which can sometimes trigger false positives in simple scanners.
- **Independent Release**: We distribute via GitHub to keep the project open and free.

**Verification**: You can audit our code right here on GitHub. We value your privacy—all voice processing happens **on your device** and never touches a server.
