# PodCards AI Engine

PodCards is an open-source AI-powered study assistant for Anki. It transforms your flashcards into interactive audio sessions, allowing you to study hands-free using state-of-the-art TTS and STT technology.

## Features

- **AI Voice Agent**: Interactive study sessions powered by Piper TTS and Whisper STT.
- **Anki Integration**: Directly syncs with your AnkiDroid database.
- **Hands-Free Study**: Answer flashcards using your voice.
- **Open Source**: Free to use, modify, and distribute.

## Getting Started

1. Clone the repository.
2. Open in Android Studio.
3. Build and run on your device.
4. Ensure AnkiDroid is installed and has API access enabled.

## Technology Stack

- **Kotlin & Jetpack Compose**: Modern Android UI.
- **Piper TTS**: Local, fast text-to-speech.
- **Whisper.cpp**: High-performance on-device speech recognition.
- **AudioTrack**: Low-latency audio streaming.
